Details.

This visualization tool uses synthetic Synthea data that mimics an early novel coronavirus (COVID-19) outbreak within the state of Massachusetts.

  • Why synthetic data? Synthetic data -- simulated data that are generated based on the trends and patterns of real data -- may provide an avenue for researchers to better understand real-world data trends without the need to overcome the obstacles involved in obtaining real patient data.
  • Main aim: We do not aim to implement an upload feature allowing users to import their own COVID-19 data. Instead, we seek to create an open exploratory visualization tool framework that allows users who have basic Python, GeoJSON and data cleaning knowledge to make visualizations from their own COVID-19 demographic data in the Observational Medical Outcomes Partnership (OMOP) common data format, and include an appropriate GeoJSON, with minimal effort.


DIY: Use your own data.

If you have COVID-19 data, real or synthetic, in the OMOP common data model format, you can use our tool to help visualize your data!

  1. Clone or download our GitHub repository
  2. To ensure that your data is properly translated for the visualization tool, use our SQL query and python script to format the data.
